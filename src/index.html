Ramda/Flikr Test project

This is an app I built to learn some new stuff

  • Functional Programming with JS (using Ramda)
  • WebPack for ES6 module loading as as a build script
  • Continuous deployment to Github Pages using TravisCI

The code is located here: https://github.com/zachlysobey/ramda-flikr

Much of the code and processes are taken directly from the following tutorials

  • Professor Frisby's Mostly Adequate Guide to Functional Programming (CH 6)
  • Various WebPack resources/tutorials including:
    • Official Getting started page
    • Foxandxss's angular-webpack-workflow
  • Auto-deploying built products to gh-pages with Travis
